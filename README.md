# Учебный проект CV

### Быстрый старт:

- Клонируйте репозиторий: `git clone https://github.com/InaKurets2021/ProjectsJS.git`
- Установите зависимости: `yarn install` или `npm install`
- Запустите проект: `yarn serve` или `npm run serve`
